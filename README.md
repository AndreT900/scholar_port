# ScholarPort

Applicazione web MERN per la gestione del portfolio accademico personale. Permette di archiviare pubblicazioni, con dettagli su autori, abstract e citazioni.

Tecnologie:
- Frontend: React
- Backend: Node.js e Express
- Database: MongoDB

## Istruzioni per l'installazione

Requisiti: Node.js installato.

### Backend
1. Apri il terminale nella cartella 'server'.
2. Esegui 'npm install' per scaricare le librerie.
3. Crea un file .env con le variabili MONGO_URI e PORT.
4. Avvia il server con 'npm start'.

### Frontend
1. Apri il terminale nella cartella 'client'.
2. Esegui 'npm install' per scaricare le librerie.
3. Avvia l'applicazione con 'npm start'.

Il browser si aprirà automaticamente all'indirizzo localhost.

## Funzionalità principali
- Gestione lista pubblicazioni.
- Ricerca per titolo o autore.
- Aggiunta, modifica ed eliminazione articoli.
- Gestione citazioni.
